Telegram Apps

Unsere Anwendungen sind Open Source und unterstützen nachvollziehbare Builds. Das bedeutet, dass jeder unabhängig überprüfen kann, dass unser Code auf GitHub genau derselbe Code ist, der für die Erstellung der Apps verwendet wurde, die man aus dem App Store oder von Google Play herunterladen kann.

Telegram Database Library (TDLib)

  • TDLib – ein plattformübergreifender Client, der die Erstellung eigener Anwendungen auf der Telegram-Plattform erleichtern soll.
  • Telegram X für Android – eine schlanke experimentelle App, basierend auf TDLib.

Inoffizielle Apps

  • Unigram, speziell für Windows optimiert (basiert auf TDLib) (Desktop und Xbox One)
  • Telegram CLI für Linux
  • MadelineProto

Quelltext

Aktuell konzentrieren wir uns auf Sachen, die für Entwickler nützlich sind. Wir bieten eine gut dokumentierte Programmierschnittstelle (API), welche es Entwicklern erlaubt, neue Telegram Apps zu erstellen und der Quelltext unserer Android, iOS, Web und Desktop Apps (Windows, macOS und Linux) sind öffentlich, genau wie die Telegram Database Library.

Dieser Code erlaubt Sicherheitsprofis die vollständige Prüfung unserer Ende-zu-Ende-Verschlüsselung. Außerdem ist eine unabhängige Verifikation möglich, dass die Telegram Apps bei Google Play und im App Store tatsächlich den gleichen Code benutzen, den wir bei Github veröffentlichen.
